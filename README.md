# desafio-fullstack-veritas

Mini Kanban (React + Go). Três colunas fixas (A Fazer, Em Progresso, Concluídas) e CRUD /tasks.

## Backend
cd backend
go run .

Rotas: GET/POST /tasks, GET/PUT/DELETE /tasks/:id, GET /health

## Frontend
cd frontend
npm install
npm run dev

Acesse http://localhost:5173
